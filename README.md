# Flare Garden Prototype Step3.0

スマートフォン横画面向けの初期プロトタイプです。
Step3.0ではプレイヤー主導の横スクロールを維持したまま、夕暮れまでの制限時間、いたずら好きなカエルと鳥、旅の結果画面を追加しています。

## 起動

このフォルダをローカルサーバーで配信して `index.html` を開きます。

```sh
python3 -m http.server 8091 --directory outputs/flare-garden
```

現在の確認用URL:

```text
http://127.0.0.1:8091/
```

## スマートフォン実機で確認

スマートフォンとこのMacを同じWi-Fiにつなぎ、スマートフォンのブラウザで以下を開きます。

```text
http://192.168.11.50:8091/
```

`127.0.0.1` はスマートフォン自身を指すため、実機確認では使えません。
MacのWi-Fiが変わると `192.168.11.50` の部分も変わることがあります。

ブラウザのタブバーやアドレスバーで画面が狭い場合は、ホーム画面に追加してから起動します。
PCやCodex内ブラウザで縦画面表示のまま確認する場合は、横向き案内の `ブラウザで続行` を押すと確認を続けられます。

iPhone Safari:

1. 共有ボタンを押す
2. 「ホーム画面に追加」を選ぶ
3. ホーム画面の `Flare Garden` アイコンから開く
4. スマートフォンを横向きにする

Android Chrome:

1. 右上メニューを開く
2. 「ホーム画面に追加」を選ぶ
3. ホーム画面の `Flare Garden` アイコンから開く
4. スマートフォンを横向きにする

PWA対応後は「アプリをインストール」と表示される場合があります。

ホーム画面から開くとブラウザUIは減ります。
完全にアプリ風の全画面表示に近づけるため、現在はWeb App ManifestとApple向けのmeta設定を入れています。

## 操作

- 移動: キャラを直接ドラッグ / 画面上の行きたい位置を押す
- 上下移動: ドラッグ時のみ少し上下へ移動
- 左移動: 左矢印 / A / 左ボタン
- 右移動: 右矢印 / D / 右ボタン
- 水やり: Space / W / 水ボタン / ブラウザでダブルクリック
- 補充: NORMALモード時の `+10 100pt` ボタン

## 夕暮れ制限

- 制限時間: 180秒
- スタート画面とカウントダウン中は時間が減りません
- プレイ開始後は、立ち止まっていても時間が進みます
- 残り時間に応じて空が昼から夕方、日没直前へ少し変化します
- 時間が0になると `夕暮れになりました` として旅の結果を表示します
- Flare Gardenへ到着した場合は、その時点でタイマーが停止します

## 水管理

- 最大水量: 50
- 開始水量: 25
- 水滴1つにつき水を1消費
- 補充コスト: 100pt
- 補充量: 10
- AUTO: 水が0になった時に、100pt以上あれば自動補充
- NORMAL: 好きなタイミングで補充ボタンから補充
- 水が0で、補充に必要な100ptも足りない場合は水やりだけできなくなります
- 水やポイントが足りない状態でも移動は継続でき、ゴールへ到達できます

## いたずら住人

- カエル: エリア2以降に出現し、近づくとプレイヤーを追いかけます
- カエルは水滴を吸収し、10滴で満腹になって報酬抽選後に退場します
- カエル報酬: なし / 水+5 / 50pt / 中花生成 / 大花生成
- 鳥: エリア3で予告表示のあと左右から飛んできます
- 鳥に接触するとゲームオーバーではなく、水を5滴こぼします

## ステージ

- ステージ全長: 12500
- ゴール位置: 12100
- 花総数: 100本
- 小花: 58本
- 中花: 28本
- 大花: 14本
- 波パターン: 20個
- 1波あたり: 5本
- カエル: 5体
- 鳥: 3羽

花はすべて固定配置です。
各波に `group` と `pattern` を持たせ、素早く小花を咲かせるか、大花へ水と時間を使うかを選べる密度へ調整しています。

## スクロール

- 自動スクロール: なし
- プレイヤー座標: `player.worldX`
- 画面上の表示位置: `player.worldX - cameraX`
- カメラ追従開始: 画面X 520
- カメラ追従終了: 画面X 760
- 進行度: プレイヤー座標から算出
- ゴール判定: プレイヤーがFlare Gardenの入口へ到達した時

カメラはプレイヤーが画面中央付近の追従範囲を超えた時だけ動きます。
左右どちらにも戻れるため、咲かせ損ねた花の位置へ戻って再挑戦できます。

## デバッグ確認

URLへ `?debug=1&metrics=1` を付けると、調整用の計測表示がCanvas内に出ます。

```text
http://127.0.0.1:8091/?debug=1&metrics=1
```

`?hitboxes=1` を付けると花の当たり判定と水滴中心も表示できます。
`?time=30` のように指定すると、確認用に制限時間を短くできます。

計測項目:

- 経過時間
- プレイヤーのワールド座標
- プレイヤーの画面座標
- カメラ座標
- 進行度
- ゴールまでの距離
- 前進距離 / 後退距離
- 残り時間と時間帯
- 画面内の花数
- カエル満腹数、吸収水量、報酬ログ
- 鳥の予告数、接触数
- 咲かせた花数 / 総数
- 小・中・大の開花数
- 使用水量
- 無駄水数
- 補充回数
- 花が画面内にない累計時間
- 花が画面内にない最長時間

## 調整箇所

- ゲームバランス: `src/config.js`
- ステージ配置: `src/stage1_layout.js`
- 画像差し替え: `src/asset_manifest.js`

`asset_manifest.js` の `null` を画像パスへ変更すると、Canvas の仮描画ではなく画像を使います。

例:

```js
characters: {
  player: "./assets/images/player-bear.png",
}
```

カエルと鳥は仮描画で実装済みです。`enemies.frog` と `enemies.bird` に画像パスを入れると差し替えられます。
